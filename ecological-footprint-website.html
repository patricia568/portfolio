The unique part about this website is that it will show the user exactly which type of ecosystem they are contributing the most waste to (forest, desert, ocean, etc...). Each ecosystem will have a unique illustration." challenge-subtitle="The Mission" challenge-content="To create an interactive experience for the user in order to motivate them enough to reduce their ecological footprint.
